## 2. Portofolio Web Development (PHP & MySQL)

Implementasi nyata dari rancangan menjadi aplikasi web dinamis yang mengintegrasikan antarmuka dengan logika server (*Back-End*) dan basis data.

*   **Teknologi:** Pure PHP, HTML5, CSS3, dan MySQL.
*   **Fitur Utama:** 
    *   Otentikasi halaman login admin menggunakan data dari database `users`.
    *   Sistem CRUD Produk (Tambah, Lihat, Edit, Hapus) lengkap dengan fitur *upload* foto ke server lokal.
    *   Manajemen input kas keuangan harian.
*   **Status Proyek:** Kode mentah (.php) sudah sukses di-upload di dalam repositori GitHub ini dan berjalan lancar di lingkungan server lokal (*localhost* XAMPP).
